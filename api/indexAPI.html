wizoscopo API getting for you daily / weekly horoscope messages from Paolo Fox or Rob Bresnzy thanks to web scraping.

Using that API it's free. So just start using me follow instructions below .

get your message


 # Here is a curl example

 curl \
 -X GET api_endpoint_url?sign=your_sign&astrologer=lastname_astrologer

GET a call to the following path endpoint :

?sign=your_sign&astrologer=lastname_astrologer

sign: Your zodiac sign in italian language.
astrologer: The lastname of your favourite astrologer. ( fox | breszny )
